# Sum List

## Motivation
List contains a set of elements maybe of same/different data type. 

## Problem Description
You are provided with a list of integers called `lst` which is hidden from you. Do not peek in the provided_code.py file! Write a python script to sum the integer items from `lst` and store it in a variable called `total`.
