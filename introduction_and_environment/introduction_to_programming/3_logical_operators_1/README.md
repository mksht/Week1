# Logical Operators 1

## Problem Description 
In the **solution.py** file, create three variables `x`, `y`, and `z`, such that:
1. `x and y == False`
2. `x or y == True`
3. `x and z == True`
4. `y or z == True`

## Testing
* To test your solution, type 'pytest' within the **solution** subdirectory
